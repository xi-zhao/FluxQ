<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| command result -> workspace latest/history files | Compare, benchmark, and doctor cross from in-memory results into shared mutable workspace JSON files. |
| baseline/export/pack command -> workspace mutable files | Baseline, export, and pack mutate `baselines/current.json`, current artifact aliases, and pack directories inside the shared workspace. |
| workspace writer -> CLI JSON output | Guarded persistence failures must surface as stable machine-readable conflict or recovery signals. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-04-01 | T | `runtime/compare.py` persistence | mitigate | Acquire the workspace lease only for compare persistence and promote `compare/latest.json` plus history via atomic replace so concurrent compares cannot interleave state. |
| T-03-04-02 | T | `diagnostics/benchmark.py` persistence | mitigate | Promote benchmark latest/history files through the atomic helpers so interrupted writes cannot leave partially overwritten benchmark JSON. |
| T-03-04-03 | T | `runtime/doctor.py` persistence | mitigate | Guard doctor latest/history writes with the same lease and atomic promotion so repeated doctor runs cannot corrupt the latest report. |
| T-03-04-04 | T | `workspace/baseline.py` and `cli.py` baseline mutation | mitigate | Persist and clear `baselines/current.json` through the workspace lease and atomic helpers so baseline set/clear cannot leave truncated or missing state under contention. |
| T-03-04-05 | T | `runtime/export.py` alias writes | mitigate | Reuse the staged alias promotion path from Plan 03 and hold the lease only for final export alias mutation so interrupted export cannot corrupt current artifacts. |
| T-03-04-06 | T | `runtime/pack.py` pack root plus history backfill | mitigate | Guard pack directory creation and history backfills with the workspace lease and atomic helpers so concurrent packaging cannot leave ambiguous partial bundle state. |
| T-03-04-07 | R | guarded write failures | mitigate | Raise the structured workspace conflict or recovery exceptions from Plan 02 so CI can attribute and retry blocked writes deterministically. |
</threat_model>
