|----------|-------------|
| `.quantum/qrun.toml` → `doctor` gate | 持久化的 provider profile 被解释为 CI 阻断结论 |
| provider auth state → JSON/JSONL payload | 外部鉴权状态被投影为 agent 可消费输出 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-02-01 | S | `run_doctor()` IBM auth resolution | mitigate | 所有 IBM access 状态都通过 `resolve_ibm_access()` 统一解析并带稳定 `reason_code`，禁止 env/saved-account 静默混用。 |
| T-09-02-02 | I | doctor JSON / JSONL payload | mitigate | 仅输出 reason codes、auth_source、instance 等非 secret 字段；测试断言无 token / Authorization / raw provider exception dump。 |
| T-09-02-03 | D | CI gate | mitigate | IBM access 异常映射到 blocking findings 与 `gate.ready = false`，避免未授权状态被误判为可提交。 |
</threat_model>
