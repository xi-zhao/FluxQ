# Support

## Support Scope

Quantum Runtime CLI is maintained as a developer tool and runtime surface for quantum workflow automation.

Good support requests include:

- bug reports
- reproducible CLI failures
- release or packaging problems
- questions about workspace layout, replay behavior, or compare guardrails
- documentation gaps

## Before Opening an Issue

- Re-run the command with `--json` when available.
- Capture the exact version from `qrun version`.
- Include the relevant command, exit code, and structured error payload.
- State whether the issue depends on optional extras such as `qiskit` or `classiq`.

## What To Include

- operating system and Python version
- install method
- exact command line
- expected behavior
- actual behavior
- minimal reproduction input when possible

## What Support Does Not Cover

- custom downstream integrations that replace the documented runtime contract
- private environment debugging without reproduction details
- sensitive vulnerability details in public threads
